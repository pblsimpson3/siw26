Saugus Iron Works 2026 Resurrection Project

I'm Robert Simpson, an instructional designer and educator. This modernization project seeks to resurrect the mid-2000s Saugus Iron Works Great Adventure Project-Based Unit (PBU). I'm using Claude.ai to update web code. All updated files are shared in a gitHub respository.

SIW26 Development Goals

  • W3C Compliant
  • Fully Accessible
  • Maintainable
  • Performance-optimized
  • Responsive design (mobile friendly)
  • Preserve 2000s aesthetic & instructional design
